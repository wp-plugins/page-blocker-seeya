This plugin allows you to block access to static pages by one or more security questions.

== Description ==

This Wordpress plugin allows you to block access to static pages by one or more security questions. First, you have to create and add some questions with answers to the main list. Answers are case insensitive. Then, associate questions from list you created with pages you need to block. After question is associated with a page - page becomes blocked until user provides correct answers on questions associated with a page, if he does - access will be granted during current session.

== Installation ==

1. Upload `pageblocker-seeya-by` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
